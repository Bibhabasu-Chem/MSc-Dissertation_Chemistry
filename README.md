# MSc Dissertation: Liquid Metal Simulation

This repository contains the Fortran source code developed for my Master's dissertation at **Banaras Hindu University (BHU)**.

## 🔬 Research Overview
The project focuses on the **Statistical Mechanical Modeling** of rare-earth liquid metal systems. The code calculates thermodynamic properties and structural factors using numerical methods.

## 🛠️ Technical Details
- **Language:** Fortran 90/95
- **Domain:** Computational Chemistry / Liquid State Physics
- **Key Features:** Numerical integration of radial distribution functions, property estimation for rare-earth systems.

## 🚀 How to Run
To compile the code using `gfortran`:
```bash
gfortran MetalTest.for -o metal_sim
./metal_sim
